## **Sobre o Projeto**

Projeto desenvolvido em 2023 para a disciplina de **Programação Orientada a Objetos** do curso de **Desenvolvimento de Sistemas** do Instituto Federal de Santa Catarina (IFSC).

O sistema **IFSC Lava Car** é uma aplicação desktop feita em **Java 18** com interface gráfica em **JavaFX**, que simula a gestão de um lava car, incluindo cadastro e gerenciamento de marcas, serviços e usuários, com persistência de dados usando banco MySQL.

---

## **Tecnologias**

- **Java 18** — Linguagem de programação principal do projeto.
- **JavaFX 21** — Framework para interface gráfica.
- **Scene Builder** — Programa usado para desenvolvimento das telas
- **MySQL 8** — Banco de dados relacional para persistência.
- **Conector JDBC MySQL 8.1.0** — Driver para conectar o Java ao banco MySQL.
